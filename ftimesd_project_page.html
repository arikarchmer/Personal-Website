Projects && hacks


"StarSpace"
My second project completed in July, '16. An entertaining web game that features you, the daring spaceship pilot, and your quest to collect massive stars before your alien counterparts do the same. Realized with Python, Javascript, HTML, CSS, Bootstrap, Ajax, Google App Engine, D3.js.

"TwitterData"
My first project completed in June, '16. A minimalist web tool for discovering twitter trends by location. TwitterData is useful for searching keywords in specific cities, or for making broad searches on a location by drawing a circle on the world map. Realized with Python, Leaflet.js, Javascript, HTML, CSS, Bootstrap, Jinja Templates, Google App Engine, and the Tweepy api for Twitter.
